Source code!

Please find attached the Lemuria Server (ability to control the Robot via Signal R), Shasta (software that runs on the Raspbery Pi), ShastaController (UWP implementation of the Raspbery Pi Controller), iOS and Android clients coming soon!

Detect the enviornment

Humans navigate using the 5 senses so lets add them to Shasta

Floor Detection

FC-51 IR chip will save the Lemurian Bot from tripping by detecting the floor everytime it moves!

Motor Driver

L239D chip should protect the Raspberry Pi from the power hungry DC motors and enable the bot's movements!

Face Detection

Using the Project Oxford by Microsoft we can convert any cheap camera to the bot's eye!

Object Avoidance

HC-SRO4 calculates the distance between the bot and an object in front and if there are none, off it goes!

Temperature Detection

Using DHT11 we can detect if the Raspberry Pi and Motor driver are over heating!

Display Panel

Small display panel is attached to the bot incase we loose signals with the shasta remote!
